ขั้นตอนที่ 1

สมัคร GitHub

01
เข้าเว็บ github.com
กด Sign up ที่มุมบนขวา
02
กรอกข้อมูล
อีเมล, รหัสผ่าน, ชื่อ username
username สำคัญมาก — จะกลายเป็น URL เว็บ: username.github.io
03
ยืนยันอีเมล
GitHub จะส่ง code มาที่อีเมล กรอกให้ครบ
04
เลือก plan: Free
ฟีเจอร์ที่ต้องใช้มีครบหมด ไม่ต้องจ่ายเงิน

สร้างเว็บแรก

01
สร้าง Repository ใหม่
กด New repository → ตั้งชื่อ username.github.io → ติ๊ก Public + Add README → Create
02
สร้างไฟล์ index.html
กด Add file → Create new file → ตั้งชื่อ index.html → วางโค้ดด้านล่าง → Commit changes
03
เปิดเว็บดู
รอ 1-2 นาที แล้วเข้า https://username.github.io
ถ้ายังไม่ขึ้น รอแป๊บแล้วลอง refresh 
<!DOCTYPE html>
<html>
<head>
  <title>บล็อกของฉัน</title>
</head>
<body>
  <h1>สวัสดี! นี่คือบล็อกแรกของฉัน</h1>
  <p>เริ่มจากศูนย์ แล้วจะพาทุกคนไปด้วยกัน</p>
</body>
</html>

ขั้นตอนที่ 3

จด Domain

ทางเลือก Domain ที่ได้ ค่าใช้จ่าย แนะนำ?
ฟรี username.github.io ฟรี ✅ แนะนำตอนนี้
ซื้อเอง ชื่ออะไรก็ได้ เช่น myblog.com ~100-300 บาท/ปี ค่อยซื้อเมื่อพร้อม

เว็บขาย Domain

เว็บข้อดีราคา
Namecheapราคาถูก มีโปรโมชั่นฟรี 1 ปี~100-200 บาท/ปี
Cloudflareราคาต้นทุน ไม่บวกกำไร~250-300 บาท/ปี
Google Domainsใช้งานง่าย เชื่อมกับ Google ได้ดี~300 บาท/ปี
ตอนนี้ยังไม่จำเป็นต้องซื้อ domain — รีบไปเขียนโพสต์แรกดีกว่า เพราะ content สำคัญกว่าชื่อเว็บมาก
